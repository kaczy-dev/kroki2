# KROKI — Krokomierz

Brutalistyczny krokomierz PWA w przeglądarce.

## Stack
- TanStack Start + React 19
- Tailwind CSS v4 (tokeny w `src/styles.css`)
- Framer Motion (animacje)
- Vite 8
- Bun (package manager)

## Zasady
- Dane wyłącznie w localStorage — bez backendu, bez konta
- Styl neo-brutalist: żółte tło, grube czarne obramowania, font Archivo Black
- Dark mode przez `data-theme="dark"` na `<html>`
- Mobile-first, PWA-ready
- Język UI: polski
